# Recursion Documentation

This folder holds the Recursion extension design, implementation-facing architecture specs, and eventual user manuals.

Start with [Recursion Extension Spec](RECURSION_EXTENSION_SPEC.md).

## Structure

- `design/` - Product, card system, and UI design documents.
- `architecture/` - Implementation-facing module and data-flow notes.
- `user/` - User-facing setup and operation guides.
- `testing/` - Verification plans and test runbooks.
