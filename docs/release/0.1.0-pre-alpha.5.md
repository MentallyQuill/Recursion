# Recursion 0.1.0-pre-alpha.5

Recursion `0.1.0-pre-alpha.5` tightens Dialogue Enhancement after live SG-1 testing showed the Dialogue pass could still return near-copy output while Prose Enhancement was making useful revisions.

## Release Highlights

- Removes the clean no-op escape hatch from the Dialogue Enhancement prompt.
- Adds dialogue-span edit ratio tracking so narration-heavy replies do not dilute dialogue changes.
- Retries exact no-op Dialogue output once, then keeps the original instead of appending or replacing with a duplicate.
- Retries under-10% dialogue-span edits once when strong slop, soft suspicion, or echoed-user context signals are present.
- Expands Dialogue slop detection with tiered strong and soft phrase detectors for forced agency, echoing, unsupported smart talk, defensive tropes, and romance cliches.
- Records Dialogue attempt, retry reason, applied lane, fallback source, whole-message edit ratio, and dialogue-span edit ratio in enhancement markers.
- Updates operator, provider, UI, and runtime-sequence docs for the new Dialogue contract.
